# OU-Pokemon-Battle-Prediction-Model
This project implements a Machine Learning model to predict the outcome of Pokémon battles in the OverUsed (OU) tier, using data parsed from Pokémon Showdown.

---

## Getting Started

### 1. Prerequisites (Git LFS)
This repository uses **Git LFS (Large File Storage)** to manage the large dataset CSV files. To ensure the datasets are downloaded correctly, please follow these steps:

1. **Install Git LFS**: Download and install it from [git-lfs.com](https://git-lfs.com/).
2. **Initialize LFS**: Open your terminal and run:
   ```bash
   git lfs install
   ```
3. **Clone the repository**:
    ```bash
    git clone [url]
    cd OU-Pokemon-Battle-Prediction-Model
    ```
4. **Pull the data (if the CSV files are just pointers)**:
    ```bash
    git lfs pull
    ```

### 2. Install Dependencies
Once the virtual environment is active, install the required libraries using pip:

 ```bash
    pip install -r requirements.txt
```
